# Brand Identity

Living Architecture brand guidelines for consistent visual identity across all products.

## Brand Overview

**Products:**
- **Living Architecture** — Extract software architecture from code as living documentation
- **Éclair** — Web-based architecture visualization platform
- **Rivière** — Architecture extraction libraries and CLI

## Design System

Three themes provide visual consistency:

| Theme | Mode | Use Case |
|-------|------|----------|
| **Stream** | Light | Default, marketing, documentation |
| **Voltage** | Dark | Developer-focused, technical contexts |
| **Circuit** | Light | Minimal, GitHub-inspired, accessibility focus |
